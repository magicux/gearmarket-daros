# Dependencias del proyecto

## Frontend
- react
- react-dom
- react-router-dom
- axios
- bootstrap
- react-hook-form
- sweetalert2

## Backend
- express
- pg
- cors
- dotenv
- jsonwebtoken
- bcryptjs
- express-validator
- nodemon

## Testing backend (para Hito 3)
- jest
- supertest

## Dependencias justificadas
- `react-router-dom`: navegación entre vistas.
- `axios`: consumo de la API REST.
- `bootstrap`: framework CSS recomendado para agilizar la interfaz.
- `react-hook-form`: manejo simple de formularios.
- `express`: construcción de API REST.
- `pg`: conexión a PostgreSQL.
- `jsonwebtoken`: autenticación basada en token.
- `bcryptjs`: hash de contraseñas.
- `express-validator`: validación de payloads.
- `jest` y `supertest`: pruebas de endpoints.
